# office-documents

Repository: https://github.com/odinlayer/ai-skills/tree/main/office-documents

License: MIT

`office-documents` is a unified document skill for:

- DOCX
- PDF
- PPTX
- XLSX

It consolidates common workflows for creation, editing, extraction, conversion,
and validation across those formats into one skill.

## Files

- `SKILL.md` - Pi-compatible unified Office document workflow.
- `README.md` - Human-facing notes and lineage.
